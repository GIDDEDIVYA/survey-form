        selectors

        Element Selector

        it is used to target the element based on the tagname

        id selector

        id selector is used to only select one element

        it represented with #

        id must be always unique

        One element can not one id

        class selector

        class Selector is used to select more than one element.

        it represented with .

        it can be duplicated

        one element can have more than one class name

        It is used to target the some portion of the element.

        • First letter pseudo element selector
        • Selection pseudo element selector
        • Before pseudo element selector
